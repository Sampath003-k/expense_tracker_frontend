# 💰 Expense Tracker (Full Stack Project)

A full-stack Expense Tracker web application built using **Spring Boot + React**.

🔗 **Live Demo:**
👉 https://expense-tracker-frontend-t6lu.onrender.com

---

## 🚀 Features

* 🔐 User Authentication (Login/Register with JWT)
* 💸 Add / Update / Delete Expenses
* 📊 Manage Projects
* 🔒 Secure API with Spring Security
* 🌐 Fully deployed (Frontend + Backend + Database)

---

## 🛠️ Tech Stack

### Frontend

* React JS
* Axios
* CSS

### Backend

* Spring Boot
* Spring Security
* JWT Authentication
* REST APIs

### Database

* PostgreSQL (Neon DB)

### Deployment

* Frontend: Render
* Backend: Render
